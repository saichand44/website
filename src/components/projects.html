Projects

Augmented Reality

Seamless integration of virtual objects into the real-world scene where users can choose the initial placement of virtual objects. As the camera moves, position and orientation dynamically update using AprilTags and P3P / PnP algorithms.

3D Reconstruction

Generating a point cloud resembling the 3D geometry of an object using a two-view stereo algorithm. Keypoint correspondences are computed using a disparity map between pairs of 2D images.

Neural Radiance Fields (NeRFs)

Synthesizing novel views of a scene using a fully-connected neural network trained on images with known camera poses. The network outputs volume density and view-dependent emitted radiance.

Object Detection & Tracking

Detecting moving objects using YOLOv3 trained on COCO dataset. Object tracking implemented using DeepSORT with feature descriptors from FaceNet to associate detections across frames.

Object Detection

Detecting football players and ball using OpenCV and traditional image processing: masking, morphological transforms, edge detection, Hough transforms. Homography used for bird's-eye-view projection.

Pick and Place Challenge

Controlling a 7-DOF Franka-Emika Panda robot using Positional Inverse Kinematics. Successfully stacked 7 blocks (4 static, 3 dynamic) — second place out of 20 teams!
